# Map building off images.

## Setup

As tested with Python 3.6 on Windows:

    py -m venv venv
    venv\Scripts\activate.bat
    pip install -r requirements.txt

## Running it

As tested with Python 3.6 on Windows:

    venv\Scripts\activate.bat
    python -m mapbuilder.run

## Tests

    venv\Scripts\activate.bat
    py.test
